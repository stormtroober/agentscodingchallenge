# Frequently Asked Questions (FAQ)

## General Questions

### What is this product?
Our product is a comprehensive platform for managing your business operations, including data processing, API integrations, and automated workflows.

### How do I get started?
1. Sign up at our website
2. Verify your email
3. Log into the dashboard
4. Create your API key
5. Follow our Integration Guide to start building

### Is there a free trial?
Yes! All new accounts receive a 14-day free trial with full access to Professional features. No credit card required.

## Account Management

### How do I reset my password?
1. Click "Forgot Password" on the login page
2. Enter your registered email address
3. Check your email for the reset link
4. Click the link and set a new password

### How do I change my email address?
1. Log into the dashboard
2. Go to Settings → Account
3. Click "Change Email"
4. Verify the new email address

### Can I have multiple users on one account?
Yes, Professional and Enterprise plans support team accounts. You can invite team members from Settings → Team Management.

## API & Technical

### What programming languages are supported?
We provide official SDKs for:
- Java
- Python
- Node.js
- Go
- Ruby

You can also use any language that supports HTTP requests.

### What are the API rate limits?
Rate limits depend on your plan:
- Basic: 60 requests/minute
- Professional: 300 requests/minute
- Enterprise: 1,000 requests/minute

### How do I handle API errors?
Check the HTTP status code and error message in the response body. Common errors:
- 401: Check your API key
- 429: You're rate limited - slow down requests
- 500: Server error - retry after a few seconds

### Is the API available 24/7?
Yes, our API has 99.9% uptime SLA. Check status.example.com for real-time status.

## Data & Security

### Is my data secure?
Yes, we take security seriously:
- All data is encrypted at rest (AES-256)
- All connections use TLS 1.2+
- Regular security audits
- SOC 2 Type II certified

### Where is data stored?
Data is stored in secure data centers in:
- US (Virginia)
- EU (Frankfurt)
- Asia Pacific (Singapore)

You can choose your primary region during signup (Enterprise only).

### Can I export my data?
Yes, you can export all your data at any time:
1. Go to Settings → Data
2. Click "Export All Data"
3. Choose format (JSON or CSV)
4. Download the generated file
